## 原生 APP 設計原則（取代網頁式思維）
- **導覽/手勢**：底部 Tab Bar、頂部 Tab 均改用原生手勢（左右滑動切換 Tab）；列表下拉刷新；次要內容用「從底部彈出的 Sheet/Modal」而非跳轉新網頁
- **輸入元件**：日期、城市、人數選擇一律使用原生 iOS 元件（輪盤式 picker、原生日期選擇器），不用網頁下拉選單（`<select>`）
- **頁面轉場**：頁面之間切換使用 iOS 風格推入/彈出動畫（push/pop、modal present），不是網頁式整頁刷新
- （待補：實際套用到 React Native 時使用的 navigation library，如 React Navigation）

## 視覺設計
- **整體風格**：活潑親切（參考 Hopper 的調性）

## 出發前詢問頁（新增，首頁之前）
- App 開啟後，先進入此頁，填完才跳轉到首頁
- **日期**：三選一切換式輸入 — 月份 / 明確日期 / 日期區間
- **城市**：選擇預計前往城市 → 顯示該城市機場清單供選擇
- **人數**：選擇乘客人數
- 三項都選完後：
  - 立即顯示「價格高或低」提示（簡單文字 + 色彩標籤，例如「這個月份偏貴」「性價比高」）
  - **價格走勢圖**：同時附上歷史價格走勢圖表（參考 Skiplagged／Hopper 做法），讓文字標籤有圖表佐證，方便挑最划算的日期
  - 顯示人數對票價、找到位子機率的提示文字
  - 完成後才進入目前的首頁（快速搜尋／外站票／反向購票 Tab）
- （待補：機場清單資料來源、價格高低判斷的資料/規則、價格走勢圖資料來源、人數與找位機率提示的具體文案）

## 搜尋欄位（三種模式）

| 欄位 | 快速搜尋 | 外站票 | 反向購票 |
|---|---|---|---|
| 機場輸入 | 出發地、目的地 | 外站出發地、出發地、目的地 | 出發地、目的地 |
| 日期 | 出發日 + 返程日 | 出發日 + 返程日 | 去程日 + 回程日 |
| 其他 | 人數、艙等 | 人數、艙等 | 人數、艙等 |

- 城市輸入支援模糊搜尋，下拉選擇 IATA 機場代碼
- 出發地／目的地可互換
- 承接「出發前詢問頁」：城市／日期／人數自動帶入預填，使用者仍可在此修改
- **★ 差異化功能：多人同行分開訂比較**：人數 ≥ 2 時，額外比較「全部人一起訂」vs「拆開分別訂」總價，較划算時提示使用者
  - （待補：拆開訂票的座位安排/劃位提醒、購買步驟說明）

## 結果頁（三種模式各自獨立）

### ① 快速搜尋結果頁
- **4 個 Tab**：最便宜 3 班 / 時間最佳 3 班 / 廉價 vs 傳統 / 鄰近日期（±7天）
- **篩選 Chips**：直飛、各航空公司（可組合）
- **排序**：價格最低 / 時間最短
- 票價包含行李費用

### ② 外站票結果頁
- 最便宜 3 個組合，每張卡顯示 **4 段完整航程 + 總價**：
  ```
  ① 外站出發地 → 出發地（接駁去程）
  ② 出發地 → 目的地（主要去程）
  ③ 目的地 → 出發地（主要回程）
  ④ 出發地 → 外站出發地（接駁回程）
  ```
- 省下多少錢（vs 傳統來回，含百分比）
- 怎麼購買說明（4 步驟）
- **接駁安全轉機時間**：系統自動計算「最短安全轉機時間」，篩掉銜接太趕的組合（參考 Kiwi.com 虛擬轉機演算法做法），避免接駁段趕不上主航班
- **★ 差異化功能：接駁段順道玩建議**：若接駁/轉機時間夠長，卡片上額外推薦該城市的順遊景點或短暫停留玩法（stopover），把接駁段包裝成「順便多玩一個城市」的賣點
  - （待補：多久轉機時間才算「夠長」、景點資料來源）

### ③ 反向購票結果頁
- 最便宜 3 個組合，每張卡顯示：
  - **去程**：正向票（出發地 → 目的地）
  - **反向回程**：購買「出發地 → 目的地」方向的票，實際只搭「目的地 → 出發地」段
- **邏輯**：系統同時搜尋正向去程 + 反向回程，找出比傳統來回票更便宜的方案
- 省下多少錢（vs 傳統來回，含百分比）
- 怎麼購買說明（4 步驟）
- ⚠️ 風險提醒（強化版）：
  - 結果頁卡片上直接顯示警告標記（不用點進去才看到）
  - 點進詳情頁後才呈現完整風險說明（放棄去程段可能導致整張票失效等），並要求使用者確認了解後才能前往訂票

## 共用功能
- 點擊航班卡片 → **詳情頁**（航段資訊、費用明細、票務資訊、里程累積、退改票規則、前往訂票按鈕）
- 底部導覽列：首頁 / 結果 / 我的
- SVG inline icons 豐富視覺
- 航班卡片顯示：航空公司 Logo、航班號、出發/抵達時間、飛行時長、轉機次數、行李費、**艙等、機型**
- 詳情頁額外顯示：**里程累積數、可否退票／改票規則**

### 「我的」頁面（新增內容）
- 會員／登入
- 收藏航班／搜尋紀錄
- 價格追蹤／降價通知
  - **邀請旅伴共同追蹤**：可將追蹤的航班分享給旅伴，雙方都收到降價通知（參考 Hopper「Watch with your travel buddy」）
- **★ 差異化功能：航空公司容忍度回報**：使用者可回報實際使用外站票／反向購票後，該航空公司是否刁難、取消里程、警告等，累積成社群評分資料庫
- （待補：各項細節功能與畫面）

### ★ 差異化功能：航空公司容忍度社群評分
- 資料來源：「我的」頁面的使用者回報
- 呈現位置：外站票／反向購票結果頁卡片上顯示該航空公司的容忍度評分（如燈號或分數），詳情頁顯示評分依據的回報摘要
- （待補：評分演算法、防止惡意/不實回報的機制、法律風險評估）

### 前往訂票按鈕 — 導轉邏輯
- 站內不完成付款，一律導去外部網站
- 導轉對象：比較「航空公司官網」與「比價/訂票平台（如 Skyscanner）」何者價格較便宜，導去較便宜的一方
- （待補：API 回傳的 deep link 格式、比價時機是查詢當下還是快取結果）
